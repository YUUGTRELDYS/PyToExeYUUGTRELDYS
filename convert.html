Convert python to exe online!

py2exe.com provides a simple way to convert python to exe online.

To convert your project, please upload it below. You can upload a single .py script as well as a zip archive of multiple files. If you are uploading multiple files in zip, please name the main file "main.py" or "run.py" and make sure it is in the root of the zip file.

You can give your exe a custom icon by putting it in a file called "icon.png" or "icon.ico" in the zip archive.

If you run into any problems, please refer to the how to guide. Join the py2exe.com discord server to get help with your project and more.

*if your project requires any python libraries, please include them in a requirements.txt file inside the zip (package names separated by newline)

The files are stored securely and are only accesible via the link that will be generated for you. All files are deleted after 2 hours. This site does not log any additional information about users
